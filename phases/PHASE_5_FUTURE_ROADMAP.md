# Phase 5: Future Roadmap

## 📌 Status: Proposed

This phase represents the planned improvements and future updates to transition **Crazy Crack** from a localized room-based application to a production-grade gaming service.

---

## 🎯 Proposed Features

### 1. Global Matchmaking & Queue System
* Allow players to queue up for quick public games.
* Automatically match players of similar win/loss ratios.
* Support for private party codes and shared links.

### 2. User Accounts & Cloud Save
* Move from local storage profile stats to a backend persistent database (e.g., MongoDB, PostgreSQL, or Firebase).
* Global player accounts with email, Google, or Apple sign-in.
* Dynamic level progression, experience points, and achievement unlock badges.

### 3. Match History & Leaderboards
* Public global leaderboards ranking players by wins, total lines completed, and win-streak multipliers.
* Personal dashboard displaying full history of past games, boards, chat summaries, and stats over time.

### 4. Interactive Power-Ups (Optional Game Modes)
* Implement alternative game boards or casual game modes.
* Custom rules (e.g., speed mode with a 10s turn limit, or wildcard spaces on grids).
* Power-ups like "Freeze" (skips an opponent's turn) or "Double Vision" (reveals a small area of an opponent's grid).

### 5. Progressive Web App (PWA) Support
* Implement service workers for offline caching, asset serving, and startup speed.
* Installable icon prompts for Android and iOS systems directly from standard web browsers.
